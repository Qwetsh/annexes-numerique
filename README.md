## Intégration en iframe

Copie ce bloc et remplace le nom de fichier par l'annexe voulue :

```html
<iframe
  src="https://qwetsh.github.io/annexes-numerique/A6_LookUp.html"
  width="100%" height="600"
  style="border:0;" loading="lazy"
  title="Annexe A6 - LookUp">
</iframe>
```

## Mettre à jour une annexe

Modifie le fichier localement, puis :

```bash
git add -A
git commit -m "maj annexe"
git push
```

GitHub Pages se redéploie automatiquement en ~1 minute.
